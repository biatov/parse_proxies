### Usage
* * *
    scrapy crawl proxies

    Example:
         ~/parse_proxies/parse_project $ scrapy crawl proxies
* * *
### Record *.sqlite and *.json
    scrapy crawl proxies -o data.json

    Example:
         ~/parse_proxies/parse_project $ scrapy crawl proxies -o quote.json
* * *
